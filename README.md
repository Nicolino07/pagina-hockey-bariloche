# Sistema de Gestión Deportiva

Proyecto para administrar y publicar información de una asociación deportiva.

## Funcionalidades
- Carga digital de planillas de partidos
- Resultados, goles, tarjetas y sanciones
- Tabla de posiciones automática
- Fixture por categoría
- Sección de noticias
- Exportación a Excel/PDF
- Sitio público para ver datos + panel administrativo

## Estructura del proyecto
- /backend — API y lógica del sistema  
- /frontend/admin — Panel de administración  
- /frontend/public — Sitio web público  

## Tecnologías 
- Backend: (Python)
- Frontend Admin: (React)
- Frontend Público: (React)
- Base de datos: PostgreSQL
- Docker para deploy
